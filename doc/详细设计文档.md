# Maze 项目详细需求与设计文档

> 本文档旨在详细描述"迷宫（Maze）"桌面应用的开发目标、功能、UI/UX 设计、技术架构及实现细节，供 AI 开发者作为唯一参考依据。
> 请 AI 在开发过程中严格遵循本文档，**如有任何需求或技术方案的不确定性，必须主动提问并等待确认，禁止自行假设或过度设计**。

## 1. 项目概述

### 1.1 项目基本信息

**项目名称**：Maze  
**项目类型**：桌面应用  
**版本号**：v1.1.0  
**开发周期**：2026-06-25 开始  

### 1.2 功能演进（v1.1 新增）

**v1.1 版本核心增强**：
- **迷宫模式**：支持正方形、圆形，预留自定义图形扩展接口
- **出入口模式**：支持边界位置、中心位置，支持用户点击指定
- **预览功能**：预览迷宫时仅渲染边界、出入口、角色，不生成完整数据
- **参数配置**：丰富的参数调整，设置后实时预览

### 1.3 技术栈

**前端框架**：
- Vue 3 (Composition API)
- TypeScript 5.x
- Vite 5.x (构建工具)

**UI 组件库**：
- PrimeVue 4.x (全局引入按需组件)
- Lucide Vue (图标库)

**状态管理**：
- Pinia 2.x

**核心依赖**：
- Electron 30.x (主进程)
- Electron Store (配置持久化)
- Canvas API (2D 绘图)
- Lucide Vue (图标库)

**开发工具**：
- ESLint + Prettier (代码规范)
- TypeScript (类型安全)
- Vitest (单元测试)

**图标库**：
- Lucide Vue 4.x (轻量级、现代图标库)

### 1.4 核心体验

生成一个居中显示、由线条墙壁构成的随机迷宫，用户使用方向键控制一个角色（黑点或自定义图标）在迷宫中移动，实时记录并展示耗时、行走长度等信息，支持保存当前迷宫参数以便下次恢复。

**v1.1 新增特性**：
- 支持多种迷宫模式（正方形、圆形）
- 支持灵活的出入口配置（边界、中心、自定义）
- 提供迷宫参数预览功能（快速预览，不生成完整数据）

### 1.5 设计理念

**极简主义**：去掉一切非必要装饰，仅保留核心信息。  
**高级感**：暗色主题、柔和阴影、微光效、玻璃态/半透明面板、无边框沉浸感。  
**高性能**：使用 Canvas API 实现流畅的 60fps 动画。  
**可维护性**：清晰的代码结构、类型安全的 TypeScript、模块化设计。  
**可扩展性**：预留自定义图形生成接口，便于未来功能扩展。

## 2. 功能需求

### 2.1 迷宫生成

#### 2.1.1 功能描述

**v1.1 新增**：支持多种迷宫模式、出入口配置、实时预览

- 随机生成一个二维迷宫
- 迷宫由线条墙壁构成（墙壁为 1px~2px 的线段，通道为空白区域）

**2.1.1.1 迷宫模式**

- **正方形模式**：
  - 标准网格结构（`rows` × `cols`）
  - 适用于经典迷宫体验

- **圆形模式**：
  - 迷宫边界为圆形
  - 内部使用网格布局，裁剪超出圆形部分
  - 适用于美观、对称的迷宫设计

- **自定义图形（预留接口）**：
  - 支持用户上传自定义形状（SVG、PNG、图片）
  - 将自定义形状转换为迷宫边界
  - 内部使用网格布局，裁剪超出形状部分
  - 为未来功能扩展预留完整接口

**2.1.1.2 出入口配置**

**入口位置**：
- **边界位置**（默认）：
  - 随机选择边界的一个位置
  - 保证入口在迷宫最外层
  - 避免与入口重叠
- **中心位置**：
  - 入口位于迷宫中心区域
  - 需确保中心区域有足够空间
- **自定义点击**：
  - 用户在画布上点击任意位置设置入口
  - 验证位置有效性（必须在迷宫区域内，且足够靠近有效路径）
  - 需要预览入口位置是否合理

**出口位置**：
- **边界位置**（默认）：
  - 随机选择与入口不重叠的边界位置
  - 确保出口在迷宫最外层
- **中心位置**：
  - 出口位于迷宫中心区域
  - 与入口位置不重叠
- **自定义点击**：
  - 用户在画布上点击任意位置设置出口
  - 验证位置有效性（必须在迷宫区域内，且足够靠近有效路径）
  - 需要预览出口位置是否合理

**出入口重叠检测**：
- 入口和出口位置不重叠
- 最小距离限制（防止过于接近）
- 出口必须可到达（算法自动验证）

**2.1.1.3 参数调整**

**迷宫参数**：
- **大小**：
  - 行数 `rows`（5-50，默认 15）
  - 列数 `cols`（5-50，默认 15）
  - 正方形模式下 `rows == cols`
  - 圆形模式下 `rows` 和 `cols` 决定尺寸

- **复杂度/分支度**：
  - 通过随机种子 `seed` 实现
  - 相同种子 + 相同参数 = 相同迷宫（可复现性）
  - 支持自定义种子输入

- **速度设置**：
  - 移动速度（1-10 单位/秒，默认 3）
  - 影响角色移动动画速度

**2.1.1.4 预览功能**

**快速预览模式**（默认开启）：
- 仅渲染迷宫边界、出入口、角色位置
- **不生成迷宫数据**（不运行生成算法）
- 渲染速度极快（< 5ms）
- 用户调整参数时实时预览
- 显示入口和出口的预览标识

**完整预览模式**：
- 渲染迷宫边界、出入口、角色位置、迷宫墙壁
- 生成完整的迷宫数据
- 渲染时间：15×15 迷宫 < 50ms
- 适合最终确认迷宫参数

**预览开关**：
- 提供预览模式切换开关
- 默认：快速预览
- 可选项：完整预览

**2.1.1.5 迷宫生成流程**

1. 用户调整参数（模式、大小、出入口等）
2. 实时预览迷宫外观（快速预览模式）
3. 用户确认或调整参数
4. 点击"生成迷宫"按钮
5. 生成完整迷宫数据（完整预览模式）
6. 初始化角色到入口位置
7. 开始计时
8. 游戏进行

**2.1.1.6 视觉标识**

- **入口标识**：
  - 绿色微光效果
  - 入口图标（箭头指向内部）
  - 入口标签："入口"

- **出口标识**：
  - 红色微光效果
  - 出口图标（箭头指向外部）
  - 出口标签："出口"

- **角色标识**：
  - 白色发光小圆点
  - 角色初始位置标注

- **边界标识**：
  - 浅色线条（2px）
  - 发光效果（可选）

#### 2.1.2 技术要求

**迷宫生成算法**：
- 使用递归回溯算法生成完美迷宫
- 确保迷宫所有单元格连通，无环，通路唯一
- 支持相同种子产生相同迷宫（可复现性）
- 生成时间在可接受范围内（15×15 迷宫应在 50ms 内完成）

**模式适配**：
- 正方形模式：标准网格处理
- 圆形模式：裁剪超出圆形的单元格
- 自定义图形：将图形轮廓转换为边界

**出入口处理**：
- 自动验证出入口位置的有效性
- 确保出入口不重叠
- 确保出口可到达
- 支持用户自定义点击设置

**预览功能**：
- 快速预览模式不生成迷宫数据，仅渲染几何元素
- 使用简化的渲染逻辑
- 实时响应参数调整（< 16ms）
- 提供性能优化（避免重复计算）

### 2.2 角色控制与动画

#### 2.2.1 功能描述

- 角色初始位于迷宫入口处
- 使用键盘方向键（↑↓←→）控制角色移动
- 角色移动必须流畅平移，采用基于 `requestAnimationFrame` 的线性插值动画，帧率稳定
- 移动速度可配置（默认每秒移动 3 个迷宫单位格）
- 角色与墙壁的碰撞检测：
  - **基于网格的精确逻辑**：角色当前所在单元格，该单元格四个方向若有墙壁则不可通行
- 达到出口时触发胜利状态：
  - 展示提示信息
  - 停止计时
  - 提供重新开始选项
- 角色外观：
  - 默认：实心黑色小圆点（直径约为单元格边长的 40%）
  - 支持自定义图标（后续扩展）

#### 2.2.2 技术要求

- 使用 `requestAnimationFrame` 实现平滑动画
- 支持 60fps 帧率
- 移动速度单位：逻辑单位（1 格 = 100 像素）
- 防止按键过快导致移动异常
- 支持多键同时按下（如果可通行）

### 2.3 数据统计与显示

#### 2.3.1 统计指标

- **计时器**：
  - 从角色第一次移动开始计时
  - 精确到 0.1 秒
  - 到达出口时自动停止
  - 格式：`MM:SS.ms`（分钟:秒.毫秒）

- **行走长度**：
  - 累计角色实际走过的**逻辑距离**
  - 每一格 = 100 逻辑单位
  - 计算每一帧位移的欧氏距离累加

- **步数**：
  - 角色每完成一次方向键指令记为一步
  - 按下方向键后角色连续移动直到松开或遇阻，只计 1 步
  - 不计算微小的像素级移动

#### 2.3.2 显示要求

- 右侧面板实时刷新上述数值
- 使用等宽字体显示数字，保持稳定
- 数值格式化显示（如 142 步、2,340 逻辑单位）

### 2.4 状态保存与恢复

#### 2.4.1 保存内容

将当前迷宫状态持久化到本地存储：

```json
{
  "mazeParams": {
    "rows": 15,
    "cols": 15,
    "seed": 123456789
  },
  "playerGridPos": {
    "row": 0,
    "col": 0
  },
  "stats": {
    "elapsed": 0,
    "distance": 0,
    "steps": 0
  },
  "settings": {
    "speed": 3,
    "theme": "dark"
  }
}
```

#### 2.4.2 保存时机

- 每次移动成功后自动保存
- 每次到达出口后自动保存
- 每次参数调整后自动保存
- 应用关闭时自动保存

#### 2.4.3 恢复逻辑

- 应用启动时尝试读取本地存储
- 若存在有效数据，恢复迷宫和角色位置
- 角色像素坐标根据恢复的网格坐标计算
- 统计数据从上次结束处继续（不重新计时）

## 3. UI/UX 设计指南

### 3.1 设计原则

**设计风格**：
- 参考Claude Code、Gemini等AI工具的橘米色简约高级风格
- 暖色调配色方案，营造专业、舒适的使用体验
- **极简主义**：去繁就简，保留核心功能，去除冗余装饰
- **层次清晰**：通过间距、阴影、颜色区分信息层级
- **微交互**：流畅的动画、细腻的反馈，提升用户体验
- **一致性**：统一的组件风格、动效节奏、色彩规范

**核心设计规范**：
- 背景色：暖米色/浅棕色调（避免纯白、纯黑）
- 主色调：暖橘色作为强调色
- 辅助色：柔和的灰色系作为中性色
- 玻璃态效果：适度的毛玻璃背景，增强层次感
- 圆角：统一使用8px-12px圆角
- 阴影：轻微的阴影，避免过重

### 3.2 布局规范

#### 3.2.1 画布区域

- 迷宫居中显示，左右自动平衡
- 画布背景：暖米色
- 迷宫边缘：微弱边框
- 墙壁线条：1.5-2px，深棕色
- 发光效果：暖橘色，柔和但不刺眼

#### 3.2.2 控制面板

- 右侧固定面板
- 半透明背景：`rgba(245, 242, 234, 0.7)`
- 模糊效果：`backdrop-filter: blur(15px)`
- 圆角：10px
- 边框：1px半透明白色/暖色
- 阴影：轻微的向上阴影，增强悬浮感

#### 3.2.3 控件规范

**按钮**：
- 尺寸：40px × 32px
- 圆角：8px
- 背景：半透明白米色，悬停时变为橘色
- 图标大小：16-18px
- 间距：8px

**滑块**：
- 尺寸：160px × 4px
- 轨道：浅灰色
- 滑块：橘色圆点，16px直径
- 刻度：5个（左右两侧）

**输入框**：
- 尺寸：120px
- 边框：1px浅灰色
- 圆角：6px
- 占位符：浅灰色

#### 3.2.4 响应式布局

- 窗口最小：900×600
- 控制面板：固定宽度280-300px
- 画布区域：自适应剩余空间
- 字体大小：自适应窗口大小（14-16px）

```typescript
// types/stats.ts

/**
 * 统计数据
 */
export interface Stats {
  /** 已用时间（毫秒） */

  /** 总行走距离（逻辑单位） */

/**
 * 格式化时间
 */
export function formatTime(ms: number): string {
  const totalSeconds = Math.floor(ms / 1000);
  const minutes = Math.floor(totalSeconds / 60);
  const seconds = totalSeconds % 60;
  const milliseconds = Math.floor((ms % 1000) / 100);

  return `${String(minutes).padStart(2, '0')}:${String(seconds).padStart(2, '0')}.${String(milliseconds).padStart(2, '0')}`;
}

/**
 * 格式化距离
 */
export function formatDistance(distance: number): string {
  return distance.toLocaleString();
}
```

## 7. 算法详解

### 9.1 递归回溯算法

#### 9.1.1 算法原理

递归回溯算法生成**完美迷宫**，确保迷宫满足以下特性：
1. **连通性**：所有单元格都可以到达
2. **无环**：没有封闭的环路
3. **唯一路径**：任意两个单元格之间只有一条路径

#### 9.1.2 算法接口

```typescript
// utils/mazeGenerator.ts

import { Cell, Maze, MazeParams, WallDirection } from '@/types/maze';

/**
 * 递归回溯算法生成迷宫
 */
export class MazeGenerator {
  /**
   * 生成迷宫
   */
  static generate(params: MazeParams): Maze {
    const { rows, cols, seed } = params;
    const grid = this.createGrid(rows, cols);
    const visitedCount = new Set<string>();

    // 设置随机种子
    const random = seed !== undefined
      ? this.createSeededRandom(seed)
      : Math.random;

    // 从入口开始
    const current = { row: 0, col: 0 };
    visitedCount.add(this.cellKey(current.row, current.col));
    grid[0][0].visited = true;

    // 栈用于回溯
    const stack: Array<{ row: number; col: number }> = [current];

    while (stack.length > 0) {
      const current = stack[stack.length - 1];
      const neighbors = this.getUnvisitedNeighbors(current.row, current.col, rows, cols, visitedCount);

      if (neighbors.length > 0) {
        // 随机选择一个邻居
        const next = neighbors[Math.floor(random() * neighbors.length)];

        // 打通当前单元格和邻居之间的墙
        this.removeWall(grid, current, next);

        // 标记为已访问
        visitedCount.add(this.cellKey(next.row, next.col));
        grid[next.row][next.col].visited = true;

        // 入栈
        stack.push(next);
      } else {
        // 没有未访问的邻居，回溯
        stack.pop();
      }
    }

    // 设置入口和出口
    const entry = { row: 0, col: 0 };
    const exit = { row: rows - 1, col: cols - 1 };

    return {
      grid,
      entry,
      exit,
      params
    };
  }

  /**
   * 创建网格
   */
  private static createGrid(rows: number, cols: number): Cell[][] {
    const grid: Cell[][] = [];
    for (let row = 0; row < rows; row++) {
      const rowArray: Cell[] = [];
      for (let col = 0; col < cols; col++) {
        rowArray.push({
          row,
          col,
          walls: {
            top: true,
            right: true,
            bottom: true,
            left: true
          },
          visited: false
        });
      }
      grid.push(rowArray);
    }
    return grid;
  }

  /**
   * 获取未访问的邻居
   */
  private static getUnvisitedNeighbors(
    row: number,
    col: number,
    rows: number,
    cols: number,
    visited: Set<string>
  ): Array<{ row: number; col: number }> {
    const neighbors: Array<{ row: number; col: number }> = [];

    // 上
    if (row > 0 && !visited.has(this.cellKey(row - 1, col))) {
      neighbors.push({ row: row - 1, col });
    }

    // 右
    if (col < cols - 1 && !visited.has(this.cellKey(row, col + 1))) {
      neighbors.push({ row, col: col + 1 });
    }

    // 下
    if (row < rows - 1 && !visited.has(this.cellKey(row + 1, col))) {
      neighbors.push({ row: row + 1, col });
    }

    // 左
    if (col > 0 && !visited.has(this.cellKey(row, col - 1))) {
      neighbors.push({ row, col: col - 1 });
    }

    return neighbors;
  }

  /**
   * 移除两个单元格之间的墙
   */
  private static removeWall(grid: Cell[][], current: { row: number; col: number }, next: { row: number; col: number }): void {
    const dx = next.col - current.col;
    const dy = next.row - current.row;

    if (dx === 1) {
      // 向右
      grid[current.row][current.col].walls.right = false;
      grid[next.row][next.col].walls.left = false;
    } else if (dx === -1) {
      // 向左
      grid[current.row][current.col].walls.left = false;
      grid[next.row][next.col].walls.right = false;
    } else if (dy === 1) {
      // 向下
      grid[current.row][current.col].walls.bottom = false;
      grid[next.row][next.col].walls.top = false;
    } else if (dy === -1) {
      // 向上
      grid[current.row][current.col].walls.top = false;
      grid[next.row][next.col].walls.bottom = false;
    }
  }

  /**
   * 单元格唯一键
   */
  private static cellKey(row: number, col: number): string {
    return `${row},${col}`;
  }

  /**
   * 创建带种子的随机数生成器
   */
  private static createSeededRandom(seed: number): () => number {
    let s = seed;
    return () => {
      s = Math.sin(s) * 10000;
      return s - Math.floor(s);
    };
  }
}
```

#### 9.1.3 复杂度分析

**时间复杂度**：O(rows × cols)  
**空间复杂度**：O(rows × cols)

#### 9.1.4 性能优化

- 使用栈代替递归，避免栈溢出
- 使用 Set 存储已访问的单元格，提高查找速度
- 使用种子随机数保证可复现性

#### 9.1.5 迷宫模式处理

**正方形模式**：
- 标准网格处理
- 入口/出口在网格范围内
- 常规迷宫生成流程

**圆形模式**：
- 创建完整的网格
- 计算圆形区域内的所有有效单元格
- 裁剪超出圆形区域的单元格
- 设置 `inValidArea: false` 标记
- 入口/出口必须在有效区域内

**自定义图形模式**（预留）：
- 接收用户上传的图形（SVG、PNG等）
- 将图形轮廓转换为迷宫边界
- 创建网格并裁剪超出部分的单元格
- 设置 `inValidArea` 标记
- 为未来功能扩展预留接口

**模式选择策略**：
- 正方形模式：适用于经典迷宫体验
- 圆形模式：适用于美观、对称的迷宫设计
- 自定义图形模式：适用于特殊场景需求

#### 9.1.6 出入口验证

**边界位置验证**：
- 检查网格坐标是否在有效范围内
- 对于正方形模式：0 ≤ row < rows, 0 ≤ col < cols
- 对于圆形模式：需要满足距离中心小于半径
- 入口和出口不能重叠
- 入口和出口之间保持最小距离（避免过于接近）

**中心位置验证**：
- 计算网格中心点（`Math.floor(rows / 2), Math.floor(cols / 2)`）
- 检查中心点是否在有效区域内
- 确保中心区域有足够空间（至少 3×3）
- 中心位置在圆形模式下需要验证

**自定义点击位置验证**：
- 验证点击位置在画布范围内
- 转换点击坐标为网格坐标
- 检查网格坐标是否在有效范围内
- 检查与边界距离是否足够（至少 1 个单元格）
- 对于圆形模式，验证点击位置是否在圆形区域内
- 生成验证结果和错误提示

**错误提示策略**：
- 参数无效时：提供清晰的错误信息
- 出入口重叠时：提示调整位置
- 位置超出范围时：提示有效范围
- 验证失败时：显示详细错误信息并高亮问题位置

```typescript
// utils/collisionDetector.ts

import { Cell, MovementDirection } from '@/types/maze';
import { Player, PlayerConfig } from '@/types/player';

/**
 * 碰撞检测器
 */
export class CollisionDetector {
  /**
   * 检测移动方向是否有墙壁阻挡
   */
  static canMove(
    playerGrid: { row: number; col: number },
    cell: Cell,
    direction: MovementDirection
  ): boolean {
    switch (direction) {
      case MovementDirection.UP:
        return !cell.walls.top;
      case MovementDirection.DOWN:
        return !cell.walls.bottom;
      case MovementDirection.LEFT:
        return !cell.walls.left;
      case MovementDirection.RIGHT:
        return !cell.walls.right;
      default:
        return false;
    }
  }

  /**
   * 检测玩家是否到达目标单元格
   */
  static reachedTarget(
    playerGrid: { row: number; col: number },
    targetGrid: { row: number; col: number }
  ): boolean {
    return (
      playerGrid.row === targetGrid.row &&
      playerGrid.col === targetGrid.col
    );
  }

  /**
   * 获取移动后的网格坐标
   */
  static getTargetGrid(
    playerGrid: { row: number; col: number },
    direction: MovementDirection
  ): { row: number; col: number } {
    switch (direction) {
      case MovementDirection.UP:
        return { row: playerGrid.row - 1, col: playerGrid.col };
      case MovementDirection.DOWN:
        return { row: playerGrid.row + 1, col: playerGrid.col };
      case MovementDirection.LEFT:
        return { row: playerGrid.row, col: playerGrid.col - 1 };
      case MovementDirection.RIGHT:
        return { row: playerGrid.row, col: playerGrid.col + 1 };
      default:
        return playerGrid;
    }
  }
}
```

### 9.3 动画循环

#### 9.3.1 帧动画

```typescript
// composables/useAnimation.ts

import { onUnmounted } from 'vue';
import { Player, MovementDirection, PlayerConfig } from '@/types/player';
import { Cell } from '@/types/maze';

/**
 * 移动动画
 */
export function usePlayerAnimation(
  player: Player,
  cellSize: number,
  cell: Cell,
  config: PlayerConfig,
  onMoveComplete?: () => void
) {
  let lastTime = 0;
  let isAnimating = false;
  let animationFrameId: number | null = null;

  /**
   * 启动动画
   */
  const startAnimation = (direction: MovementDirection) => {
    if (player.isMoving || !player.targetX || !player.targetY) {
      return;
    }

    player.direction = direction;
    player.isMoving = true;
    lastTime = performance.now();
    isAnimating = true;

    animate();
  };

  /**
   * 动画循环
   */
  const animate = (currentTime: number = performance.now()) => {
    if (!isAnimating) {
      return;
    }

    const deltaTime = currentTime - lastTime;
    lastTime = currentTime;

    // 计算步长（逻辑单位/秒）
    const speed = config.speed; // 逻辑单位/秒
    const stepSize = (speed * deltaTime) / 1000;

    // 计算移动距离
    const dx = player.targetX - player.x;
    const dy = player.targetY - player.y;
    const distance = Math.sqrt(dx * dx + dy * dy);

    // 检查是否已到达
    if (distance <= stepSize) {
      // 到达目标
      player.x = player.targetX;
      player.y = player.targetY;
      player.isMoving = false;
      player.direction = null;
      isAnimating = false;

      if (animationFrameId) {
        cancelAnimationFrame(animationFrameId);
        animationFrameId = null;
      }

      onMoveComplete?.();
      return;
    }

    // 移动
    const moveX = (dx / distance) * stepSize;
    const moveY = (dy / distance) * stepSize;
    player.x += moveX;
    player.y += moveY;

    animationFrameId = requestAnimationFrame(animate);
  };

  /**
   * 停止动画
   */
  const stopAnimation = () => {
    isAnimating = false;

    if (animationFrameId) {
      cancelAnimationFrame(animationFrameId);
      animationFrameId = null;
    }
  };

  // 组件卸载时清理
  onUnmounted(() => {
    stopAnimation();
  });

  return {
    startAnimation,
    stopAnimation
  };
}
```

```typescript
// utils/mazeRenderer.ts

import { Maze, Cell, Theme } from '@/types/maze';

/**
 * 迷宫渲染器
 */
export class MazeRenderer {
  /**
   * 渲染迷宫
   */
  static render(
    canvas: HTMLCanvasElement,
    maze: Maze,
    cellSize: number,
    offsetX: number,
    offsetY: number,
    theme: Theme
  ): void {
    const ctx = canvas.getContext('2d');
    if (!ctx) return;

    const { rows, cols, grid, entry, exit } = maze;

    // 清空画布
    ctx.fillStyle = theme.background;
    ctx.fillRect(0, 0, canvas.width, canvas.height);

    // 渲染入口
    this.renderEntry(ctx, entry.row, entry.col, cellSize, offsetX, offsetY, theme);

    // 渲染出口
    this.renderExit(ctx, exit.row, exit.col, cellSize, offsetX, offsetY, theme);

    // 渲染墙壁
    this.renderWalls(ctx, grid, cellSize, offsetX, offsetY, theme);

    // 渲染角色（如果有）
    this.renderPlayer(ctx, maze.player, cellSize, offsetX, offsetY, theme);
  }

  /**
   * 渲染墙壁
   */
  private static renderWalls(
    ctx: CanvasRenderingContext2D,
    grid: Cell[][],
    cellSize: number,
    offsetX: number,
    offsetY: number,
    theme: Theme
  ): void {
    ctx.strokeStyle = theme.wallColor;
    ctx.lineWidth = 2;
    ctx.lineCap = 'round';

    // 设置发光效果
    ctx.shadowBlur = 5;
    ctx.shadowColor = theme.wallGlow;

    const startX = offsetX;
    const startY = offsetY;

    for (let row = 0; row < grid.length; row++) {
      for (let col = 0; col < grid[row].length; col++) {
        const x = startX + col * cellSize;
        const y = startY + row * cellSize;
        const cell = grid[row][col];

        if (cell.walls.top) {
          ctx.beginPath();
          ctx.moveTo(x, y);
          ctx.lineTo(x + cellSize, y);
          ctx.stroke();
        }

        if (cell.walls.right) {
          ctx.beginPath();
          ctx.moveTo(x + cellSize, y);
          ctx.lineTo(x + cellSize, y + cellSize);
          ctx.stroke();
        }

        if (cell.walls.bottom) {
          ctx.beginPath();
          ctx.moveTo(x, y + cellSize);
          ctx.lineTo(x + cellSize, y + cellSize);
          ctx.stroke();
        }

        if (cell.walls.left) {
          ctx.beginPath();
          ctx.moveTo(x, y);
          ctx.lineTo(x, y + cellSize);
          ctx.stroke();
        }
      }
    }

    // 重置阴影
    ctx.shadowBlur = 0;
  }

  /**
   * 渲染入口
   */
  private static renderEntry(
    ctx: CanvasRenderingContext2D,
    row: number,
    col: number,
    cellSize: number,
    offsetX: number,
    offsetY: number,
    theme: Theme
  ): void {
    const x = offsetX + col * cellSize;
    const y = offsetY + row * cellSize;

    ctx.fillStyle = theme.entryGlow;
    ctx.beginPath();
    ctx.arc(x + cellSize / 2, y + cellSize / 2, cellSize / 3, 0, Math.PI * 2);
    ctx.fill();
  }

  /**
   * 渲染出口
   */
  private static renderExit(
    ctx: CanvasRenderingContext2D,
    row: number,
    col: number,
    cellSize: number,
    offsetX: number,
    offsetY: number,
    theme: Theme
  ): void {
    const x = offsetX + col * cellSize;
    const y = offsetY + row * cellSize;

    ctx.fillStyle = theme.exitGlow;
    ctx.beginPath();
    ctx.arc(x + cellSize / 2, y + cellSize / 2, cellSize / 3, 0, Math.PI * 2);
    ctx.fill();
  }

  /**
   * 渲染角色
   */
  private static renderPlayer(
    ctx: CanvasRenderingContext2D,
    player: any, // 假设玩家有 x, y 属性
    cellSize: number,
    offsetX: number,
    offsetY: number,
    theme: Theme
  ): void {
    const x = offsetX + player.x;
    const y = offsetY + player.y;

    // 发光效果
    ctx.shadowBlur = 10;
    ctx.shadowColor = theme.playerGlow;

    // 绘制角色
    ctx.fillStyle = theme.playerColor;
    ctx.beginPath();
    ctx.arc(x, y, cellSize * 0.2, 0, Math.PI * 2);
    ctx.fill();

    // 重置阴影
    ctx.shadowBlur = 0;
  }
}
```

```vue
    props.maze,

    props.offsetX,
    props.offsetY,
    props.theme

  render();
  window.addEventListener('resize', render);

  if (animationFrameId) {

  }
  window.removeEventListener('resize', render);

  render();
});
```

```typescript
// tests/unit/mazeGenerator.test.ts
import { MazeGenerator } from '@/utils/mazeGenerator';
import { Maze, MazeParams } from '@/types/maze';

describe('MazeGenerator', () => {
  describe('generate', () => {
    it('应该生成有效的迷宫', () => {
      const params: MazeParams = { rows: 5, cols: 5 };
      const maze = MazeGenerator.generate(params);

      expect(maze.grid.length).toBe(5);
      expect(maze.grid[0].length).toBe(5);
      expect(maze.entry).toEqual({ row: 0, col: 0 });
      expect(maze.exit).toEqual({ row: 4, col: 4 });
    });

    it('相同种子应该产生相同迷宫', () => {
      const params1: MazeParams = { rows: 10, cols: 10, seed: 12345 };
      const maze1 = MazeGenerator.generate(params1);

      const params2: MazeParams = { rows: 10, cols: 10, seed: 12345 };
      const maze2 = MazeGenerator.generate(params2);

      // 比较网格结构（简化比较）
      expect(maze1.grid.length).toBe(maze2.grid.length);
      expect(maze1.entry).toEqual(maze2.entry);
      expect(maze1.exit).toEqual(maze2.exit);
    });

    it('迷宫应该是完美的（无环）', () => {
      const params: MazeParams = { rows: 5, cols: 5 };
      const maze = MazeGenerator.generate(params);

      // 检查所有单元格都是连通的
      const visited = new Set<string>();
      const stack = [{ row: maze.entry.row, col: maze.entry.col }];
      visited.add(`${maze.entry.row},${maze.entry.col}`);

      while (stack.length > 0) {
        const current = stack.pop()!;
        const cell = maze.grid[current.row][current.col];

        if (!cell.walls.top && current.row > 0) {
          const key = `${current.row - 1},${current.col}`;
          if (!visited.has(key)) {
            visited.add(key);
            stack.push({ row: current.row - 1, col: current.col });
          }
        }

        if (!cell.walls.right && current.col < maze.cols - 1) {
          const key = `${current.row},${current.col + 1}`;
          if (!visited.has(key)) {
            visited.add(key);
            stack.push({ row: current.row, col: current.col + 1 });
          }
        }

        if (!cell.walls.bottom && current.row < maze.rows - 1) {
          const key = `${current.row + 1},${current.col}`;
          if (!visited.has(key)) {
            visited.add(key);
            stack.push({ row: current.row + 1, col: current.col });
          }
        }

        if (!cell.walls.left && current.col > 0) {
          const key = `${current.row},${current.col - 1}`;
          if (!visited.has(key)) {
            visited.add(key);
            stack.push({ row: current.row, col: current.col - 1 });
          }
        }
      }

      // 所有单元格都应该被访问
      for (let row = 0; row < maze.rows; row++) {
        for (let col = 0; col < maze.cols; col++) {
          expect(visited.has(`${row},${col}`)).toBe(true);
        }
      }
    });
  });
});
```
