# Maze - 桌面迷宫应用

一个基于 Electron + Vue 3 + TypeScript 构建的极简、高级的桌面迷宫应用。

## 🌟 核心特性

- **多种迷宫模式**：支持正方形、圆形、自定义图形模式
- **智能出入口配置**：支持边界、中心、自定义点击位置
- **快速预览功能**：实时预览迷宫，不生成数据
- **面向对象设计**：清晰的类封装，易于维护
- **橘米色简约风格**：参考 AI 工具的专业舒适设计
- **流畅动画**：60fps Canvas 渲染
- **多主题支持**：暖色、海洋、森林三套主题
- **完整错误提示**：友好的验证机制

## 🛠️ 技术栈

- **前端**：Vue 3 (Composition API) + TypeScript 5.x + Vite 5.x
- **状态管理**：Pinia 2.x
- **UI 组件库**：PrimeVue 4.x + Lucide Vue
- **桌面框架**：Electron 30.x
- **绘图引擎**：Canvas API

## 📋 功能模块

### 迷宫生成
- 递归回溯算法生成完美迷宫
- 支持 3 种迷宫模式（正方形/圆形/自定义）
- 灵活的出入口配置
- 快速预览模式

### 角色控制
- 键盘方向键控制
- 基于网格的碰撞检测
- 平滑动画，60fps
- 可配置移动速度

### 数据统计
- 实时计时（0.1s 精度）
- 步数统计
- 行走距离统计
- 自动保存状态

### UI/UX
- 橘米色简约风格
- 玻璃态面板
- 多主题切换
- 流畅微交互

## 📁 项目结构

```
maze/
├── electron/              # Electron 主进程
├── src/                   # Vue 前端
│   ├── components/        # Vue 组件
│   ├── views/            # 视图组件
│   ├── store/            # Pinia 状态管理
│   ├── utils/            # 工具函数
│   ├── types/            # TypeScript 类型定义
│   ├── constants/        # 常量配置
│   ├── composables/      # 组合式函数
│   └── assets/           # 资源文件
├── tests/                # 测试文件
├── doc/                  # 项目文档
├── package.json
├── tsconfig.json
├── vite.config.ts
└── electron-builder.json
```

## 🎨 设计理念

- **极简主义**：去繁就简，保留核心功能
- **层次清晰**：通过间距、阴影、颜色区分信息层级
- **微交互**：流畅的动画、细腻的反馈
- **一致性**：统一的组件风格和动效节奏
