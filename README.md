# Maze - 桌面迷宫应用

一个基于 Electron + Vue 3 + TypeScript 构建的极简、高级的桌面迷宫应用。

## 🌟 特性

- **多种迷宫模式**：支持正方形、圆形、自定义图形模式（预留接口）
- **智能出入口配置**：支持边界位置、中心位置、自定义点击设置
- **快速预览功能**：实时预览迷宫外观，不生成数据，提升用户体验
- **面向对象设计**：清晰的类封装，易于维护和扩展
- **橘米色简约风格**：参考 AI 工具的专业、舒适设计
- **流畅动画**：基于 Canvas API 的 60fps 平滑动画
- **多主题支持**：暖色、海洋、森林三套主题
- **完整错误提示**：完善的验证机制和友好的错误提示

## 🛠️ 技术栈

- **前端框架**：Vue 3 (Composition API)
- **UI 组件库**：PrimeVue 4.x + Lucide Vue
- **状态管理**：Pinia 2.x
- **桌面框架**：Electron 30.x
- **编程语言**：TypeScript 5.x
- **构建工具**：Vite 5.x
- **绘图引擎**：Canvas API

## 📋 功能模块

### 1. 迷宫生成
- 递归回溯算法生成完美迷宫
- 支持正方形、圆形、自定义图形模式
- 灵活的出入口配置（边界、中心、自定义）
- 快速预览功能

### 2. 角色控制
- 键盘方向键控制移动
- 基于网格的精确碰撞检测
- 60fps 平滑动画
- 可配置移动速度

### 3. 数据统计
- 实时计时器（0.1秒精度）
- 步数统计
- 行走距离（逻辑单位）
- 状态持久化

### 4. UI/UX
- 橘米色简约风格
- 玻璃态面板设计
- 多主题切换
- 流畅的微交互

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 打包应用

```bash
npm run package
```

## 🎨 设计理念

- **极简主义**：去繁就简，保留核心功能
- **层次清晰**：通过间距、阴影、颜色区分信息层级
- **微交互**：流畅的动画、细腻的反馈
- **一致性**：统一的组件风格和动效节奏
