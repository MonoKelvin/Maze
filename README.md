## ✨ 项目亮点

- 🎨 **高级简约大气UI设计** - 参考Gemini/ClaudeCode风格
- 🎯 **完美迷宫算法** - 递归回溯，保证迷宫质量
- 🎮 **流畅的动画效果** - 60fps，丝滑体验
- 🎨 **4种精美主题** - 暖色、海洋、森林、紫色

- ⌨️ **键盘操作** - 方向键和WASD，操作便捷
- 📊 **实时统计** - 用时、步数、距离

## 🚀 快速开始

### 环境要求
- Node.js >= 16.0
- npm >= 8.0

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
npm run electron:dev
```

## 🎮 游戏指南

### 基本操作

| 操作 | 按键 |
|------|------|
| 上移 | ↑ 或 W |
| 下移 | ↓ 或 S |
| 左移 | ← 或 A |
| 右移 | → 或 D |
| 重新生成 | 点击按钮 |
| 重置游戏 | 点击按钮 |
| 重置设置 | 点击按钮 |

### 游戏流程
1. **生成迷宫** - 点击"重新生成"按钮
2. **调整参数** - 拖动滑块调整大小、速度
3. **开始游戏** - 使用键盘控制角色移动
4. **到达出口** - 到达出口自动胜利

## 💡 功能特性

### 🎨 精美主题

| 主题 | 描述 | 色调 |
|------|------|------|
| **暖色**（默认） | 橘米色风格 | 温暖、舒适 |
| **海洋** | 蓝青色风格 | 清新、冷静 |
| **森林** | 绿色风格 | 自然、生机 |
| **紫色** | 紫色风格 | 优雅、神秘 |

### ⚙️ 迷宫参数

- **大小**：5×5 到 50×50 可调
- **速度**：1-10 单位/秒
- **种子**：自定义或随机
- **预览**：快速预览/完整生成

### 📊 统计系统

- ⏱️ **用时**：精确到0.1秒
- 👣 **步数**：每次指令计一步
- 📏 **距离**：累计行走距离

## 🛠️ 技术栈

### 前端框架
- **Vue 3** - Composition API
- **TypeScript** - 类型安全
- **Vite 5** - 构建工具
- **Pinia 2** - 状态管理

### UI组件
- **PrimeVue 4** - UI组件库
- **Lucide Vue** - 图标库

### 核心技术
- **Electron 30** - 桌面应用
- **Canvas API** - 2D绘图
- **CSS Variables** - 主题系统

## 🎯 设计理念

### 极简主义
去除一切非必要装饰，保留核心功能。

### 高级感
暗色主题、柔和阴影、微光效、玻璃态。

### 面向对象
所有逻辑封装到类中，减少全局函数。

### 可维护性
清晰的代码结构、类型安全、模块化设计。

### 可扩展性
预留接口，方便未来功能扩展。

## 🎨 UI设计特点

### 玻璃态效果
- 半透明背景：`rgba(255, 255, 255, 0.5)`
- 模糊效果：`backdrop-filter: blur(20px)`
- 圆角边框：8-12px
- 细边框：1px半透明

### 微交互
- 按钮悬停：背景色变化 + 阴影
- 移动动画：requestAnimationFrame 60fps
- 胜利效果：脉冲光环动画

### 字体规范
- 标题：Inter/Inter-Regular, 16px
- 正文：Inter/Inter-Regular, 14px
- 数字：JetBrains Mono, 16px等宽
